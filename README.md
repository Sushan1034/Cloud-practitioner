# Cloud-practitioner
This repo consist of the learning materials and guide for the cloud practitioner examination.

# What is DevOps ?

DevOps is the process where code changes are run,tested,deploy in a flexible    way so that software delivery could be faster.

# 5 DevOps tools 
 -Git
 -Github
 -Jenkins
 -Containers
 -Kubernetes
 -AWS
 
# What is CI?
CI(Continuous integration) is the practice of automatically building and testing of code every time changes are pushed to shared repository.

# Common CI Tools
 -Jenkins
 -Github Actions
 -GitLab CI
 -Azure Devops Pipeline

# CI important
 -Helps finds bugs
 -Prevents it works in my machine problems
 -Improves code quality
 -Speeds up deployment

# What is CD?

CD(Continuous Deployment) is the practice of automatically deploying tested code to production or staging environment.

# Common CD tools
 -Jenkins
 -GitHub Actions
 -GitLab CI
 -AWS Devops
 -AWS CodePipeline

# Important 
 -Faster Releases
 -Less manual work
 -Few deployment errors
 -Continuos Feedback

# Simple CI/CD pipeline flow

CI -> Build -> Test -> CD -> Deploy
